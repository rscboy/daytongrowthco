# daytongrowthco

## Tech stack

- Next.js App Router
- React
- Turbopack for local development and production builds
- TypeScript
- Tailwind CSS
- Vercel

## Commands

- `npm run dev` starts Next.js with Turbopack.
- `npm run build` regenerates Markdown alternates and runs `next build --turbopack`.
- `npm run typecheck` runs TypeScript without emitting files.
- `npm run lint` runs ESLint.
